# Suivre l'avancement d'une requête

## Accéder à la liste de mes requêtes

Depuis le menu latéral : **Mes requêtes**

Vous y trouvez toutes vos requêtes avec leur statut actuel.

---

## Cycle de vie d'une requête

```
[Soumise] → [En cours d'examen] → [En traitement] → [Résolue] → [Clôturée]
                                         ↓
                               [En attente d'information]
                                         ↓
                               [En traitement] (reprise)
```

| Statut | Signification |
|---|---|
| 🟡 **Soumise** | Requête reçue, non encore prise en charge |
| 🔵 **En cours d'examen** | Un agent examine votre dossier |
| 🟠 **En attente d'information** | Un agent vous a demandé des précisions |
| 🟣 **En traitement** | Des actions sont en cours |
| 🟢 **Résolue** | La requête a été traitée, en attente de votre validation |
| ⚫ **Clôturée** | Requête terminée |

---

## Consulter le détail d'une requête

Cliquez sur une requête pour accéder à sa **fiche détaillée**, qui contient :

- Le **fil de discussion** avec l'agent
- L'**historique des statuts**
- Les **pièces jointes** échangées
- La **date de création** et la **date de dernière modification**

---

## Répondre à une demande d'information

Si votre requête passe en statut **En attente d'information** :

1. Ouvrez la fiche de la requête
2. Consultez le message de l'agent dans le fil de discussion
3. Répondez via le champ **Ajouter un commentaire**
4. Joignez tout document demandé si nécessaire
5. Cliquez sur **Envoyer**

La requête repasse automatiquement en **En traitement**.

---

## Filtrer et rechercher

Dans la liste de vos requêtes, utilisez les filtres disponibles :

- **Par statut** : Ouverte / En cours / Clôturée
- **Par urgence**
- **Par date** : ordre croissant ou décroissant
- **Recherche** : par mots-clés dans le titre ou la description
