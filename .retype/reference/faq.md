# Foire aux questions

## Adhérents

!!!
**Je n'ai pas reçu l'e-mail d'invitation**

Vérifiez votre dossier **Spam / Courrier indésirable**. Si l'e-mail n'est pas présent, contactez votre agent référent ou écrivez à [support@syndicat.org](mailto:support@syndicat.org).
!!!

!!!
**Puis-je soumettre plusieurs requêtes simultanément ?**

Oui. Il n'y a pas de limite au nombre de requêtes actives. Nous vous recommandons toutefois de regrouper les problèmes liés dans une seule requête.
!!!

!!!
**Ma requête est en attente depuis longtemps, que faire ?**

Si votre requête est en statut **Soumise** depuis plus de 5 jours ouvrés, ajoutez un commentaire pour signaler l'urgence ou contactez directement votre agent référent.
!!!

!!!
**Puis-je modifier une requête après soumission ?**

Non. Une fois soumise, la requête ne peut plus être modifiée. Vous pouvez en revanche ajouter des informations complémentaires via le fil de discussion.
!!!

!!!
**Comment annuler une requête ?**

Ouvrez la requête, cliquez sur **Options → Annuler la requête**. Cette action est irréversible.
!!!

---

## Agents

!!!
**Comment réassigner une requête à un autre agent ?**

Depuis la fiche requête, cliquez sur **Réassigner** et sélectionnez l'agent destinataire. Une notification lui sera envoyée automatiquement.
!!!

!!!
**Puis-je traiter des requêtes hors de ma catégorie ?**

Oui, si votre rôle le permet. Consultez l'administrateur si vous n'avez pas accès à certaines catégories.
!!!

---

## Technique

!!!
**L'application ne charge pas correctement**

Videz le cache de votre navigateur (Ctrl+Shift+R) et réessayez. Si le problème persiste, contactez le support.
!!!

!!!
**L'application est-elle accessible sur mobile ?**

Oui, le portail est responsive et fonctionne sur smartphone et tablette via le navigateur mobile.
!!!
