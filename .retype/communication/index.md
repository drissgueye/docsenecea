# Communication syndicale

## Consulter la section Communication syndicale

Pour accéder à la section **Communication syndicale**, sélectionnez le menu **« Communication »** dans le menu latéral gauche. Vous serez alors redirigé vers l’espace dédié à la communication interne de la plateforme.

<img src="/assets/c1.png" width="100%">

---

## 🔔 Notifications

Une fois dans la section Communication, vous avez accès à l’ensemble des notifications.

En fonction de votre rôle, vous ne verrez que les notifications qui vous concernent. Chaque notification peut être consultée en détail et peut vous rediriger vers la requête associée afin d’en suivre l’évolution.

---

## 📣 Diffusions envoyées

En sélectionnant l’onglet **« Diffusions envoyées »**, vous accédez à l’historique des messages déjà diffusés sur la plateforme, tels que :

* Emails ;
* Messages WhatsApp ;
* Communications internes.

---

## ✉️ Nouvelle diffusion

Pour envoyer une nouvelle communication, cliquez sur le bouton **« Nouvelle diffusion »** situé en haut à droite de l’écran.

Vous devez ensuite :

* Choisir le canal de diffusion (**Email** ou **WhatsApp**) ;
* Remplir le formulaire de message ;
* Ajouter éventuellement une pièce jointe ;
* Sélectionner les destinataires.

<img src="/assets/c2.png" width="100%">

Les destinataires peuvent être :

* Les membres d’un bureau exécutif ;
* Les entreprises ;
* Les délégués ;
* Les pôles ;
* Ou l’ensemble des utilisateurs de la plateforme.

<img src="/assets/c3.png" width="100%">

---

## 👁️ Aperçu et envoi

Avant l’envoi, vous pouvez cliquer sur le bouton **« Aperçu »** pour visualiser le rendu final du message.

Une fois vérifié, cliquez sur **« Envoyer la diffusion »** pour transmettre le message aux destinataires sélectionnés.

!!!info
L’envoi de diffusion est soumis à vos droits d’accès. Certains utilisateurs peuvent avoir des restrictions selon leur rôle.
!!!
