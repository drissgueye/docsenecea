#  Bureaux Executifs 


## voir la liste des bureaux


Cette section permet de consulter et de gérer les différents bureaux exécutifs enregistrés sur la plateforme.

Pour accéder à cette fonctionnalité, sélectionnez le menu **« Bureau Exécutif »** dans le menu latéral. Vous y trouverez la liste complète des bureaux créés ainsi que les informations relatives à leur composition et à leur organisation.

Selon votre rôle et les privilèges qui vous sont attribués, vous pourrez effectuer les actions suivantes :

* Consulter la liste des bureaux exécutifs ;
* Visualiser les détails d'un bureau ;
* Créer un nouveau bureau ;
* Modifier les informations d'un bureau existant ;
* Supprimer un bureau ;
* Ajouter ou retirer des membres d'un bureau ;
* Consulter la liste des membres associés à chaque bureau.

Chaque bureau est composé de plusieurs membres exerçant des responsabilités spécifiques au sein du Syndicat. Cette fonctionnalité permet de centraliser les informations relatives à la gouvernance et d'assurer une gestion efficace des différents organes de direction.

## Créer, modifier ou supprimer un bureau

Selon les droits qui vous sont accordés, vous pouvez créer, modifier ou supprimer un bureau exécutif depuis cette section.
<img src="/assets/b3.png" width="100%"> 
<div style="margin-top:20px;"></div> 

### Créer un bureau

Pour créer un nouveau bureau :

1. Cliquez sur le bouton **« Nouveau bureau »** ;
2. Renseignez les informations demandées dans le formulaire ;
3. Cliquez sur **« Enregistrer »** pour valider la création du bureau.

### Modifier un bureau

Pour modifier les informations d'un bureau existant :

1. Sélectionnez le bureau concerné dans la liste ;
2. Cliquez sur le bouton **« Modifier »** ;
3. Effectuez les modifications souhaitées ;
4. Cliquez sur **« Enregistrer »** pour sauvegarder les changements.

### Supprimer un bureau

Pour supprimer un bureau :

1. Sélectionnez le bureau à supprimer ;
2. Cliquez sur le bouton **« Supprimer »** ;
3. Confirmez l'opération lorsque la fenêtre de confirmation s'affiche.

!!!warning
La suppression d'un bureau peut entraîner la perte des informations et associations qui lui sont liées. Assurez-vous de disposer des autorisations nécessaires avant d'effectuer cette opération.
!!!


## Ajouter ou supprimer des membres d'un bureau

Cette fonctionnalité permet de gérer les membres associés à un bureau exécutif.

### Ajouter un membre

Pour ajouter un nouveau membre à un bureau :

1. Sélectionnez le bureau concerné dans la liste ;
2. Cliquez sur le bouton **« Ajouter un membre »** ;
3. Sélectionnez le membre à intégrer au bureau ;
4. Validez l'opération pour enregistrer l'ajout.

Le nouveau membre sera alors automatiquement associé au bureau sélectionné.

### Supprimer un membre

Pour retirer un membre d'un bureau :

1. Sélectionnez le bureau concerné ;
2. Accédez à la liste des membres associés ;
3. Cliquez sur l'icône **Supprimer** (🗑️) située en face du membre à retirer ;
4. Confirmez l'opération dans la fenêtre de confirmation.

Le membre sera alors retiré du bureau tout en conservant son compte utilisateur sur la plateforme.

!!!info
La suppression d'un membre d'un bureau n'entraîne pas la suppression de son compte utilisateur. Seule son association avec le bureau concerné est supprimée.
!!!
