# Administration

Cette section est destinée aux **administrateurs** ainsi qu’aux utilisateurs disposant du rôle **Admin** ou **SG** du portail.

---

## 🔐 Accès à l’espace administration

L’espace d’administration est accessible depuis le menu latéral en sélectionnant **« Administration »**.

L’accès est strictement réservé aux comptes disposant des rôles autorisés (**Admin** ou **SG**).

---

## 👤 Gestion des utilisateurs

Accédez à **Administration → Utilisateurs** pour gérer les comptes utilisateurs de la plateforme.

### Actions disponibles :

* Créer un compte utilisateur (adhérent ou agent) ;
* Modifier les informations d’un utilisateur ;
* Suspendre ou supprimer un compte ;
* Réinitialiser le mot de passe d’un utilisateur.

---

## 🧩 Gestion des rôles

Accédez à **Administration → Rôles** pour gérer l’attribution des rôles aux utilisateurs.

### Fonctionnalités :

* Consulter la liste des utilisateurs et leurs rôles attribués ;
* Attribuer un rôle à un utilisateur via le bouton **« Attribuer un rôle »** ;
* Sélectionner l’utilisateur, le rôle et la portée du rôle dans le formulaire ;
* Supprimer un rôle attribué en cliquant sur l’icône de suppression associée.

---

## 📊 Liste des rôles et permissions

| Rôle               | Permissions                                                                             |
| ------------------ | --------------------------------------------------------------------------------------- |
| **Adhérent**       | Soumettre et suivre ses propres requêtes                                                |
| **Délégué**        | Créer des requêtes pour son entreprise, suivre les requêtes et participer au traitement |
| **Chef de Pôle**   | Recevoir les requêtes de son pôle, les analyser et les transmettre aux membres de pôle  |
| **Membre de Pôle** | Traiter les requêtes et mettre à jour leur statut                                       |
| **Administrateur** | Accès complet à la plateforme (gestion des utilisateurs, paramètres, rôles, etc.)       |

---

## 🏢 Gestion des pôles

Accédez à **Administration → Pôles** pour gérer les structures organisationnelles.

### Actions disponibles :

* Créer un pôle ;
* Modifier les informations d’un pôle ;
* Suspendre ou supprimer un pôle.

---

## 🏬 Gestion des compagnies

Accédez à **Administration → Compagnies** pour gérer les entreprises enregistrées sur la plateforme.

### Actions disponibles :

* Créer une compagnie ;
* Modifier les informations d’une compagnie ;
* Suspendre ou supprimer une compagnie.

---

!!!info
Les actions disponibles dans cette section dépendent des droits attribués à votre compte utilisateur.
!!!
